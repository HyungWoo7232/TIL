# Branch 명령어

* 브랜치 생성

```bash
% git branch 브랜치명
```

* 브랜치 목록

```bash
% git branch
```

* 브랜치 이동

```bash
% git checkout 브랜치명
(브랜치명) %
```

* 브랜치 생성 및 이동

```bash
% git checkout -b 브랜치명
(브랜치를 생성하면서 이동!!-> 제일 많이 활용)
```

* 브랜치 병합

```bash
(master) % git merge 브랜치명
```

* 브랜치 삭제

```bash
% git branch -d 브랜치명
```
